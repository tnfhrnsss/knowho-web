## About Knowho

**Knowho** is an iOS app that lets you save and identify phone numbers outside of your contacts — displayed as caller ID using iOS's Call Blocking & Identification extension.

- Save numbers with custom names
- Identify unknown callers in the Phone app
- All data stored locally on device — nothing sent to servers
- Supports iOS 16.0 and later

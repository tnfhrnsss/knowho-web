핵심 요약

Knowho는 어떠한 개인정보도 수집하거나 외부로 전송하지 않습니다. 모든 데이터는 오직 사용자의 기기 내부에만 저장됩니다.

1. 수집하는 개인정보

Knowho는 어떠한 개인정보도 수집하지 않습니다.

사용자가 입력한 이름 및 전화번호는 오직 사용자의 기기 내부에만 저장되며, 외부 서버나 개발자에게 전송되지 않습니다.

2. 데이터 저장 방식

  • 모든 데이터는 기기 로컬 저장소(App Groups, UserDefaults)에 저장됩니다
  • iCloud 동기화를 사용하지 않습니다
  • 서버, 데이터베이스, 클라우드 서비스를 운영하지 않습니다

3. 제3자 데이터 공유

Knowho는 어떠한 제3자와도 사용자 데이터를 공유하지 않습니다.

광고 네트워크, 분석 도구, 외부 API를 사용하지 않습니다.

4. 전화 권한 사용

Knowho는 Apple의 공식 Call Directory Extension API를 사용하여 전화 수신 시 발신자 이름을 표시합니다.

이 과정에서 실제 통화 내용이나 통화 기록에는 접근하거나 저장하지 않습니다.

5. 아동 개인정보 보호

Knowho는 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 개인정보 처리방침 변경

본 방침이 변경될 경우 앱 업데이트를 통해 공지합니다. 중요한 변경사항은 앱 내 알림으로도 안내드립니다.

Summary

Knowho does not collect or transmit any personal information. All data is stored solely on the user's device.

1. Information We Collect

Knowho does not collect any personal information.

Names and phone numbers entered by the user are stored solely on the user's device and are never transmitted to any external server or to the developer.

2. Data Storage

  • All data is stored locally on your device (App Groups, UserDefaults)
  • iCloud sync is not used
  • We do not operate any servers, databases, or cloud services

3. Third-Party Sharing

Knowho does not share any user data with third parties.

We do not use advertising networks, analytics tools, or external APIs.

4. Phone Permission Usage

Knowho uses Apple's official Call Directory Extension API to display caller names when receiving calls.

We do not access or store any actual call content or call history.

5. Children's Privacy

Knowho is not directed at children under the age of 14, and we do not knowingly collect personal information from children.

6. Changes to This Policy

If this policy changes, we will notify users through an app update. Significant changes will also be communicated via in-app notifications.
